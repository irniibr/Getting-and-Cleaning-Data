#Code Book for Course Project

##Variables Used

1. Data 
  - x_test, 
  - y_test, 
  - subject_test, 
  - x_train, 
  - y_train, and
  - subject_train

2. Labels 
  - activity_labels contain labels to be used for labelling y_test and y_train data

3. Features
  - features contains list of column names to be used for x_test and x_train

4. Mean and standard deviation
  - meanstd is used to extract mean and standard deviation only

5. Merge data
  -test_data contains merged data from test folder
  -train_data contains merged data from train folder
  -mergedata is the merged data from tes_data and train_data
